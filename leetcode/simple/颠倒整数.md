## 题目描述
> 给定一个范围为 32 位 int 的整数，将其颠倒。

## 示例

```
输入: 123
输出: 321

输入: 120
输出: 21

输入: -123
输出: -321
```
## 代码

```
/**
 * @param {number} x
 * @return {number}
 */
var reverse = function(x) {
    let flag = true;
    let arr = [];
    let sum = 0;
    x = x + '';
    flag = x[0] == '-';
    if (flag) {
        x = x.slice(1)
    }
    arr = x.split('');
    for (let i = 0; i < arr.length; i++) {
        sum += arr[i] * Math.pow(10, i);
    }
    if (sum > 2147483648) return 0;
    return flag ? -sum : sum;
};
```
