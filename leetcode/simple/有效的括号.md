## 题目描述
> 给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。
括号必须以正确的顺序关闭，"()" 和 "()[]{}" 是有效的但是 "(]" 和 "([)]" 不是。

## 代码

```
/**
 * @param {string} s
 * @return {boolean}
 */
var isValid = function(s) {
    const hash = {
        '(': ')',
        '[': ']',
        '{': '}'
    }
    const arr = [];
    for (let i = 0; i < s.length; i++) {
        if (hash[s[i]]){
            arr.push(s[i])    
        } else {
            if (!(hash[arr.pop()] == s[i])) {
                return false;
            }
        }
    }
    if (arr.length) {
        return false;
    } else {
        return true;
    }
};
```
