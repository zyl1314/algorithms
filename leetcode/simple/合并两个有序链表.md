## 题目描述
> 合并两个已排序的链表，并将其作为一个新列表返回。新列表应该通过拼接前两个列表的节点来完成。 

## 示例

```
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```
## 代码

```
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} l1
 * @param {ListNode} l2
 * @return {ListNode}
 */
var mergeTwoLists = function(l1, l2) {
    if (l1 == null) return l2;
    if (l2 == null) return l1;
    let head = null;
    let first = null;
    while(l1 && l2) {
        if(l1.val <= l2.val) {
            if (first) head.next = l1;
            head = l1;
            l1 = l1.next;
        } else {
            if (first) head.next = l2;
            head = l2;
            l2 = l2.next;
        }
        !first && (first = head);
    }
    if (l1) head.next = l1;
    if (l2) head.next = l2;
    return first;
};
```
